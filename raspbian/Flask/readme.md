#### * URI (Uniform Resource Identifier) 통합 자원 식별자  
  인터넷에 있는 자원을 나타내는 유일한 주소. 하위개념으로 URL, URN이 있다.  
#### * URL (Uniform Resource Locator) 파일 자원 식별자  
  네트워크상에서 자원이 어디 있는지 알려주기 위한 규약. 웹사이트 주소로 알고 있지만, URL은 웹 사이트 주소뿐만 아니라 컴퓨터 네트워크상의 자원을 모두 나타낼 수 있다.  
#### * HTTP Method  
- GET: 데이터를 가져올 때  
  서버에서 정보를 조회하거나 특정 리소스를 가져올 때 주로 사용한다.
  데이터를 URL 뒤에 붙여서 전송한다.
  데이터가 주소창에 그래로 노출되어 보안에 취약하며, 전송할 수 있는 데이터 양에 제한이 있다.  
- POST: 데이터를 보낼 때
  서버에 새로운 데이터를 생성하거나 업데이트하는 작업을 수행할 때 사용한다.
  URL이 아닌 HTTP 요청 본문에 담아서 보낸다.  
  데이터가 겉으로 드러나지 않아 GET 보다 보안성이 상대적으로 높으며, 대용량 데이터 전송이 적합하다.
