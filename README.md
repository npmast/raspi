# raspi
www.raspberrypi.com > software > Raspberry Pi Imager Download for Windows  
원하는 OS 를 설치할 수 있다.
### 대표 운영체제  
#### 1. 라즈비안: Raspberry Pi 를 위한 최적화 OS  
Debian  
GPIO: RPi.GPIO  
Broadcom/RP1 칩셋에 완전히 튜닝됨.  
GPIO, 카메라, VPU, 하드웨어 가속등이 바로 사용 가능  
카메라/OpenCV 사용에 최적  
메모리 사용 가벼움  
커널도 Pi 전용 패치  
=> 하드웨어 제어 + 안정성 중심  
#### 2. 우분투: 표준 개발 환경  
Ubuntu  
GPIO: libgpiod  
x86/ARM 공통 환경 - 서버/클라우드/ROS2와 동일  
최신 패키지, Snap, Docker, systemd 완전 활용  
카메라/OpenCV 사용이 제한적  
Pi 전용 기능을 일부 제한  
메모리 사용 무거움  
=> 개발/서버/ROS 중심
